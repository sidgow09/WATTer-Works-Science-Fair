## 📝 Project Overview
**WATTer works** is an environmental engineering project aimed at promoting sustainability by reclaiming wasted energy from existing infrastructure. The goal of this project is to reuse the flow of sewer water within pipes to generate clean electricity. 

By utilizing custom-designed 3D-printed turbine blades and a specialized pipe attachment, the system captures the kinetic energy of moving wastewater to spin a motor, effectively converting fluid flow into measurable electrical energy.

### How It Works:
1. **Flow Capture:** Wastewater moving through the pipe passes through a custom-engineered pipe attachment.
2. **Kinetic Conversion:** The force of the water spins a set of custom, hydrodynamic 3D-printed blades.
3. **Power Generation:** The spinning blades turn the shaft of an attached motor, which acts as a generator to produce electricity.
4. **Measurement:** Output is monitored using a multimeter to track voltage and current generation.
